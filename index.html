System Architecture

Complete Microsoft 365 + Claude AI integration blueprint for Bexel Support.

📧 1. Email Ingestion — Microsoft 365
  • Emails to [email protected] trigger Power Automate
  • Flow auto-creates ticket in SharePoint List
  • Auto-acknowledgement sent to client immediately
  • Email thread ID stored for reply threading
📋 2. Database — SharePoint Online
  • Single SharePoint List: BexelSupportTickets (22 columns)
  • Document Library for email attachments
  • Full audit trail with versioning enabled
  • Power BI connects here for live dashboards
🤖 3. AI Layer — Claude API
  • Summarise ticket from subject + description
  • Suggest resolution from knowledge base history
  • Draft professional client reply emails
  • Auto-classify priority and category
  • Search past solutions for similar issues
⚡ 4. Automation — Power Automate
  • Email → SharePoint ticket (Standard license)
  • Status change → Teams notification (Standard)
  • Resolved → Save to KB + send reply (Premium)
  • Daily digest report to manager via Teams
📊 5. Dashboards — Power BI + This App
  • This app: live ticket management for agents
  • Power BI: SLA trends, volume, resolution time
  • Embed as SharePoint page or Teams Tab
  • Manager view: escalation heatmap, workload
🧠 6. Knowledge Base — Auto-growing
  • Every resolved ticket's solution auto-saved
  • Claude searches KB before assigning new tickets
  • Same SharePoint list — no extra infrastructure
  • Future: AI generates monthly FAQ from KB
